# Frontend — ServiceNow AI Automation Platform

React 19 single-page application. See the [root README](../README.md) for full documentation.

## Scripts

| Command | Description |
|---------|-------------|
| `npm start` | Dev server on http://localhost:3000 |
| `npm run build` | Production build → `build/` |
| `npm test` | Run tests |

## Environment Variables

| Variable | Description |
|----------|-------------|
| `REACT_APP_API_URL` | Backend API base URL (default: `http://localhost:3001/api`) |

For production set this in Vercel dashboard or in `frontend/.env.production`.
